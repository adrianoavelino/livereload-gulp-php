Livereload com Gulp e PHP
==================

Dependências:
- node / npm
- git
- gulp global

Clone o repositório:    
```
git clone https://github.com/adrianoavelino/livereload-gulp-php.git
```
Entre no repositório:
```
cd livereload-gulp-php
```

Instale as dependências:
```
npm install

```
Instale o gulp globalmente:
```
npm install -g gulp
```
Inicie o gulp
```
gulp
```
